# ToDoApp

## О проекте
**ToDoApp** — это iOS-приложение для управления списком задач.  
Приложение позволяет создавать, редактировать, удалять и отмечать задачи как выполненные.  
Также реализован поиск по задачам и сортировка по дате создания/редактирования.

Проект выполнен в рамках тестового задания на позицию iOS-разработчика.


## Основной функционал

- Просмотр списка задач
- Создание новой задачи
- Редактирование существующей задачи
- Отметка задачи как выполненной / невыполненной
- Удаление задач
- Поиск задач по названию
- Контекстное меню с быстрыми действиями: редактировать, удалить, поделиться
- Темный UI, гибкий интерфейс
- Счетчик задач внизу экрана
- кнопка "Сброс" для очистки CoreData и возврата приложения в исходное состояние



## Технологии и инструменты

- Swift 6 / UIKit  
- CoreData для хранения задач  
- VIPER архитектура  
- GCD (многопоточность) для работы с CoreData и API  
- Xcode 16 / macOS  
- Unit-тесты с использованием XCTest



## Архитектура

Приложение реализовано с использованием **VIPER**:  

- **View** — отвечает за отображение данных и взаимодействие с пользователем  
- **Interactor** — бизнес-логика и работа с CoreData / сетью  
- **Presenter** — связывает View и Interactor, форматирует данные  
- **Router** — навигация между экранами  
- **Entity** — модель данных (`ToDoItem`)
